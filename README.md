# hospital-management-system
This is a Flask based web application for manage Hospital database. 
It allows the Admin, Doctors, and Patients to interact with the system based on their roles so that they can book appointments without scheduling conflicts, manage and track patient history/medical records and manage user data(by admin).
The application is built using HTML, CSS/Bootstrap and Jinja2 for frontend development and Flask as backend framework. For creation of database SQLite was used. This app allows users to perform different tasks based on their role. 
Run the app by running app.py file
